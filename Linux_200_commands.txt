1) Help / Discover commands (Ubuntu)
man / --help

Read manual / help.

man grep
grep --help

apropos

Search man pages by keyword.

apropos "copy file"

which / type

Find where a command is coming from.

which python3
type ls

whereis

Find binary + man page locations.

whereis nginx

2) Files & Directories
pwd

Print current directory.

pwd

ls

List files.

ls
ls -l
ls -la
ls -lh
ls -lt

cd

Change directory.

cd /etc
cd ..
cd ~
cd -     # go back

mkdir

Create directory.

mkdir test
mkdir -p a/b/c

rmdir

Remove empty directory.

rmdir emptydir

touch

Create empty file / update timestamp.

touch a.txt

cp

Copy files/folders.

cp a.txt b.txt
cp -r dir1 dir2
cp -i a.txt b.txt   # ask before overwrite

mv

Move / rename.

mv a.txt b.txt
mv *.log logs/

rm

Delete file/folder.

rm a.txt
rm -r folder/
rm -f a.txt
rm -rf folder/      # dangerous if wrong path

ln (links)

Hard link / symbolic link.

ln a.txt a_hardlink
ln -s /var/log/syslog syslog_link

file

Detect file type.

file script.sh

stat

Detailed file info.

stat a.txt

3) View & Read Files
cat

Print file content.

cat a.txt

less (best viewer)

Scroll big files safely.

less /var/log/syslog

head / tail

Top/bottom lines.

head -n 20 a.txt
tail -n 50 a.txt
tail -f app.log      # live logs

nl

Show with line numbers.

nl a.txt

wc

Count lines/words/bytes.

wc a.txt
wc -l a.txt

4) Search (very important)
grep

Search text.

grep "error" app.log
grep -i "error" app.log
grep -n "error" app.log
grep -r "password" .
grep -v "INFO" app.log

find

Find files by rule (real filesystem search).

find . -name "*.log"
find /var/log -type f -size +50M
find . -type f -mtime -7

locate (fast)

Search using database (need updatedb sometimes).

locate nginx.conf
sudo updatedb

5) Text Processing (DevOps logs)
sort
sort names.txt
sort -n numbers.txt

uniq

Works best after sort.

sort list.txt | uniq
sort list.txt | uniq -c

cut

Cut columns.

cut -d: -f1 /etc/passwd

tr

Replace characters.

echo "a,b,c" | tr ',' ' '

sed

Search/replace stream editor.

sed 's/dev/prod/g' file.txt
sed -n '1,20p' file.txt

awk

Field-based processing.

awk '{print $1}' file.txt
awk -F: '{print $1,$3}' /etc/passwd

xargs

Turn input into arguments.

cat files.txt | xargs rm -f

6) Redirection & Pipes (must know)
> overwrite, >> append, | pipe
echo "hello" > a.txt
echo "again" >> a.txt
cat a.txt | grep hello

tee (show + save)
echo "log line" | tee -a output.log

sudo chown arfat:arfat file.txt
sudo chgrp docker file.txt

umask
